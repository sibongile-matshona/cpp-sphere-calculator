# C++ Sphere Calculator

This program calculates the volume and radius of a sphere based on user input.

This project was completed as part of a Computer Engineering course at the University of North Texas.

## Features
- Uses mathematical formulas for volume and radius
- Accepts user input
- Performs calculations using floating point values
- Displays formatted output

## What I learned
- How to apply formulas in code
- How to work with floating point numbers
- How to structure calculations in a program

## How to run
Compile and run using a C++ compiler:

g++ sphere_calculator.cpp  
./a.out
